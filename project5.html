Branded Editions Landing Page Templates

A modular design system for scalable international publisher landing pages.

Role: Visual Design Intern | Tools: HTML, CSS, JavaScript, Bootstrap, Swiper.js, Trello

Context

The Branded Editions team at PressReader supports publishers in building their own branded mobile and desktop news reading experiences. As a part of this initiative, we needed a scalable solution for creating customized landing pages for publishers that were visually consistent, brand-aligned, and easy to deploy.

I was responsible for creating four reusable HTML/CSS templates that allowed for layout flexibility while maintaining PressReader’s overall brand identity.

Tech Stack & Workflow

The templates were built using HTML, CSS, Bootstrap 4, and integrated JavaScript libraries like Swiper.js for touch-enabled carousels and smooth scrolling effects. I prioritized reusable code, modularity, and semantic HTML to ensure future updates would be easy for any dev to follow.

To facilitate adoption by other team members, I created a Trello card documenting usage instructions and required image assets. Additionally, I created a device mockup asset and example screenshots to clearly demonstrate visual structure.

Outcome

The final template system streamlined our production process and made it easier for the team to launch localized publisher pages quickly and consistently.
